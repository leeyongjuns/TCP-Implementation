TCP Protocol Implementation (C)
Overview

This project implements a simplified TCP protocol from scratch in C.

The goal is to understand how TCP works internally, including connection establishment, data transmission, and connection termination.

Features
TCP packet structure implementation
3-way handshake (connection establishment)
4-way handshake (connection termination)
Data transmission (send/recv)
TCP state machine
Sliding window mechanism
Kernel-level packet handling simulation

Tech Stack
C
Socket Programming
Linux Environment

Key Concepts
TCP State Machine (CLOSED, LISTEN, SYN_SENT, ESTABLISHED, etc.)
Reliable data transmission
Flow control
Packet encapsulationㅍ
